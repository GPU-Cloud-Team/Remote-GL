Framework of gRemote

gRemote is the first open-source framework for remote-resource sharing and isolation in the GPU cloud, as well as offering an elastic and heterogeneous GPU resource pool. Essentially, gRemote designs a light-weight, out-of-box virtualization on application level, independent from operating systems and hardware. Internally, gRemote disaggregates traditional computing-rendering functionalities from server-side into loosely-coupled well-isolated components across the high-speed network such as RDMA. Moreover, to reduce the network bandwidth usage, we propose a novel Command Characteristics-Oriented (CCO) compression algorithm by taking advantage of the characteristics of OpenGL commands. Finally, we propose Resource Throttle with Negotiation-oriented Scheduler to support a better isolation and scalability.
